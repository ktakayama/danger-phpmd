# danger-phpmd

[Danger](http://danger.systems/ruby/) plugin for [phpmd](https://phpmd.org/).

## Installation

    $ gem install danger-phpmd

## Usage

<blockquote>Run phpmd and send warn comment.
<pre>
phpmd.binary_path = "vendor/bin/phpmd"
phpmd.run ruleset: "rulesets.xml"
</pre>
</blockquote>
